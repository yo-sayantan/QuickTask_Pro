PROJECT ASSIGNMENT: QuickTask
=============================

1. OBJECTIVE
------------
QuickTask is a simplified task management application designed to organize tasks efficiently. This project aims to introduce students to mobile development using **Flutter** and backend integration with **Back4App**. Students will build a functional app that supports essential task management features.

2. CORE FEATURES
----------------
### A. User Authentication
*   Implement secure user authentication using Back4App.
*   Enable functionality for users to Sign Up and Log In.

### B. Task Management
*   Enable users to Add, View, and Delete tasks.
*   Ensure each task includes a **Title** and a **Due Date**.

### C. Task Status
*   Allow tasks to be marked as **Completed** or **Pending**.
*   Provide a mechanism to toggle the status of existing tasks.

3. OPTIONAL FEATURES (Bonus)
----------------------------
*   **Edit Task**: Allow modification of task details (Title, Due Date).
*   **UI Enhancements**: Apply styling improvements for a polished user experience.
*   **Error Handling**: gracefully handle network or validation errors.

4. PROJECT STRUCTURE
--------------------
*   **Frontend**: Develop the UI using Flutter, focusing on a clean and minimalistic design.
*   **Backend**: Configure Back4App to manage User and Task data classes.
*   **Integration**: Connect Frontend and Backend to perform CRUD operations.

5. SUBMISSION REQUIREMENTS
--------------------------
1.  **Code Repository**:
    *   Create a private GitHub repository.
    *   Upload all Flutter project files and configuration assets.
2.  **Demonstration**:
    *   Record a 1-minute video demonstrating the working application.
    *   Upload to YouTube (Unlisted) and share the link.
3.  **Submission**:
    *   Submit the GitHub repository link and the video link.

6. TIMELINE & ASSESSMENT
------------------------
*   **Deadline**: 11 May 2024
*   **Assessment Criteria**:
    *   Functional completeness (Auth, Task CRUD).
    *   Code organization, readability, and structure.
    *   Successful Frontend-Backend integration.
    *   Adherence to submission guidelines.

CONCLUSION
----------
QuickTask provides a practical opportunity to master Flutter and Back4App interactions by building a real-world application scenario.
